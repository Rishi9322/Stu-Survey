🧪 Testing Workflow

Recommended Testing Order:

  1. Authentication: Start with login to get auth token
  2. User Management: Test CRUD operations for users (admin only)
  3. Surveys: Create, read, update, delete surveys
  4. Survey Responses: Submit survey responses as student
  5. Analytics: View analytics and statistics
  6. Complaints: Submit and manage complaints
  7. AI Features: Test AI chat and insights
  8. Utility: Test debug and markdown endpoints
⚠️ Important Notes:
  • Ensure Apache server is running on port 80
  • Database must be properly configured
  • Always authenticate before testing protected endpoints
  • Admin endpoints require admin role authentication
  • Test data will be inserted into the database
